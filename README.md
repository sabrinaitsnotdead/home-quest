# 🏡 Home Quest

Home Quest é um app de gamificação da rotina doméstica criado por mim, para mim.

## Por que criei isso?

Tenho TDAH e executar tarefas do dia a dia é genuinamente difícil — não por preguiça, mas porque o cérebro com TDAH precisa de estímulo e recompensa para iniciar e manter ações repetitivas e pouco interessantes. Limpar a pia, dobrar roupa, descer o lixo: tarefas simples que viram obstáculos enormes quando falta esse gatilho.

A solução que encontrei foi transformar a rotina em jogo. Em vez de uma lista de afazeres que acumula culpa, o Home Quest torna cada tarefa uma missão com pontos, níveis e conquistas. A ideia é simples: se o cérebro precisa de dopamina pra agir, que a dopamina venha junto com a tarefa.

## O que o app faz

- ✅ Lista de tarefas organizadas por dia da semana
- 🔄 Hábitos diários com streak (sequência de dias)
- ⚡ Sistema de XP e níveis conforme as tarefas são concluídas
- 🏆 Conquistas desbloqueáveis
- ➕ Tarefas personalizadas com periodicidade (semanal, quinzenal, mensal ou diária)
- 📊 Resumo mensal com histórico de desempenho

## Como usar

É um arquivo HTML único — sem instalação, sem cadastro, sem internet obrigatória. Basta abrir no navegador. O progresso fica salvo localmente no dispositivo.

Também está disponível como página web via GitHub Pages.

## Tecnologias

HTML, CSS e JavaScript puros. Nenhuma dependência externa além das fontes do Google Fonts.
